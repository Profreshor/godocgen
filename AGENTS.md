# AI Agent Rules

## Non-Negotiable

AI agents are **not permitted to write source code** for this project.

This includes, but is not limited to:

- Go code
- Tests
- Shell scripts
- Templates
- Configuration files
- SQL
- HTML/CSS/JavaScript
- GitHub Actions
- Build scripts

The human author writes every line of code.

---

## Allowed Contributions

AI agents may:

- Explain concepts.
- Explain standard library packages.
- Explain third-party libraries.
- Discuss architecture.
- Review designs.
- Identify bugs from provided code.
- Suggest improvements.
- Explain compiler or runtime errors.
- Produce pseudocode.
- Help break problems into smaller pieces.
- Review APIs and documentation.
- Discuss tradeoffs.

---

## Required Coaching Style

When helping:

- Start with high-level guidance.
- Prefer questions over answers.
- Use pseudocode instead of implementation.
- Increase specificity only if the author is genuinely stuck.
- Never replace the learning process with generated code.

The objective is to teach, not to autocomplete.
